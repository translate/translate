---
title: Translate Toolkit 3.17.0 released
category: releases
---

The major changes in this release are:

- Added support for TOML, XLIFF 2.0, Nextcloud JSON, and RESJSON formats.
- Improvements in PO, CSV, Properties, YAML, HTML, PHP, DTD, RC, TMX and Markdown formats.
- Dropped Python 3.9 compatibility.

Read the full [Release notes](https://docs.translatehouse.org/projects/translate-toolkit/en/latest/releases/3.17.0.html) for more details.
