title: Translate Toolkit 3.18.0 released
category: releases

The major changes in this release are:

- Improvements in the PO, XLIFF, TXT, RC, Properties and HTML formats.
- Removed Directory and ZIPFile storages
- Dropped support for Python 3.10

Read the full [Release notes](https://docs.translatehouse.org/projects/translate-toolkit/en/latest/releases/3.18.0.html) for more details.
